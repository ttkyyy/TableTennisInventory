## Базы данных/Задание

### Работа с реляционной БД

- Выбрать тематическую область на усмотрение практиканта, лучше выбирать наиболее близкую и знакомую область. Например, из хобби.
- Создать новую БД в SQL Server Management Studio.
- Создать несколько таблиц в БД на выбранной предметной области, не менее 5, с основными полями:
  - строка;
  - десятичное число;
  - дата и время;
  - `guid`;
  - булевый тип.
- Сделать связи между БД:
  - `1:N`;
  - `N:N`.
- Сделать уникальный индекс на любой таблице. Быть готовым рассказать, почему именно на этом поле индекс уместен и какие минусы в связи с этим возможны.
- Научиться экспортировать и импортировать БД через бэкапы и SQL-дампы.
- Написать запросы на SQL, текст запросов выложить на GitHub:
  - выборка данных с фильтрацией, сортировкой;
  - удаление, изменение данных;
  - выборка с группировкой;
  - выборка из нескольких связанных таблиц (`LEFT JOIN`, `RIGHT JOIN`, `INNER JOIN`).
